# Weather Analysis

This project performs weather analysis using Python, pandas, and matplotlib.

## Prerequisites

- Python 3.x
- Jupyter Notebook or VS Code (with Jupyter extension)

## Requirements

Install the necessary dependencies using pip:

```bash
pip install pandas requests matplotlib jupyter
```

## Usage

### Running in VS Code
1. Open this folder in VS Code.
2. Open the [Untitled.ipynb](Untitled.ipynb) file.
3. Select a Python kernel in the top-right corner.
4. Click **Run All** to execute the analysis.

### Running in Jupyter Notebook
1. Open your terminal in the project directory.
2. Run the following command:
   ```bash
   jupyter notebook
   ```
3. Open [Untitled.ipynb](Untitled.ipynb) in the browser that opens.
4. Go to **Cell > Run All**.

## Project Structure

- [Untitled.ipynb](Untitled.ipynb): Main analysis notebook.
- `.ipynb_checkpoints`: Jupyter notebook checkpoints (auto-save files).
